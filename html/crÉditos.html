Información general sobre este recurso educativo
Título PROYECTO DE ASESORAMIENTO HISTÓRICO PARA LA CREACIÓN DE UN VIDEOJUEGO
Descripción Proyecto en grupo destinado a 1º de la ESO en el que el alumnado debe crear la base o beta de un videojuego de temática histórica una vez realizada una recopilación de datos históricos, seleccionado personajes históricos o inventado tramas o personajes ficticios, todo ello haciendo uso de herramientas digitales. Cada miembro del grupo deberá asumir un rol en la búsqueda de información y creación de contenidos audiovisuales. Una vez ideado ese modelo de videojuego de temática histórica (no violento, preferentemente educativo) deben trasladar su idea una presentación (Canva, Genially...) para poder posteriormente exponerlo en clase haciendo uso de las pizarras digitales. Deberán explicar todo el proceso y en qué consiste la propuesta de videojuego. Este proyecto tiene diferentes objetivos: 1. Motivar para conseguir del alumnado una plena inmersión en la evolución histórica. 2. Fomentar el trabajo colaborativo y desarrollar la destreza para hablar en público. 3. Búsqueda de información, edición, ordenamiento y creación a través del uso de herramientas digitales. 4. Nociones básicas de emprendimiento laboral.
Autoría Víctor Manuel Maíllo Chicano
